Font Style

CSS dapat mengatur style dari teks yangada pada halaman webyang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur pertaan teks, value yang dapat digunakan yaitu left, right, center.
  • color: digunakan unutk memberikan warna pada teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: mengatur jarak antar kata.
